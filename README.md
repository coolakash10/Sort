# Sort
Implementing 6 different types of sorting.

Two types of sorting are available

A) Internal Sorting
B) External Sorting

A)Internal Sorting :
    It has always some subtypes
    
    a) Selection 
      1. Straight Selection Sort
      2. Heap Sort
    b) Insertion
      1. Straight Insertion Sort
      2. Shell Sort
    c) Exchange
      1. Bubble Sort
      2. Quick Sort
      
B) External Sorting
  1) Merge Sort
